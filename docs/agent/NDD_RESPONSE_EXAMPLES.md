# NDD Response Examples

## Example: Memory Concern

User:

```md
My dad is repeating the same questions. Is this dementia?
```

Assistant:

```md
Repeating the same questions can be a sign of memory difficulty, but it does not automatically mean dementia.

**What it may suggest**
- **Short-term memory issue:** Repeated questions can happen when someone is not retaining recent information.
- **Multiple possible causes:** Stress, poor sleep, medications, depression, infections, and neurological conditions can all affect memory.
- **Pattern matters:** It is more concerning if the behavior is new, worsening, or interfering with daily life.

**What to do next**
- Write down examples, dates, and how often it happens.
- Note any medication changes, illness, sleep changes, or mood changes.
- Consider scheduling an evaluation with a primary care doctor or neurologist.
```

## Example: Parkinson's Symptom

User:

```md
Does a hand tremor mean Parkinson's?
```

Assistant:

```md
A hand tremor can happen in Parkinson's disease, but it can also have many other causes.

**What matters**
- **Timing:** Parkinson's tremor often appears at rest, but not all tremors follow that pattern.
- **Other symptoms:** Slowness, stiffness, smaller handwriting, balance changes, or reduced arm swing can matter clinically.
- **Evaluation:** A neurologist can assess the tremor pattern and look for other signs.

A tremor alone is not enough to confirm Parkinson's.
```

## Example: Urgent Neurological Symptoms

User:

```md
My mom suddenly can't speak clearly and one side of her face is drooping.
```

Assistant:

```md
This could be urgent. Sudden trouble speaking and facial drooping can be signs of a stroke.

Call emergency services or seek emergency care now.
```
