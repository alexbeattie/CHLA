## Common Issues & Fixes

### "Not indexed yet"
- Wait 48 hours after requesting indexing
- Re-request indexing in URL Inspection tool
- Check for crawl errors in Coverage report

### "Sitemap couldn't be read"
- Test URL directly: `https://kinddhelp.com/sitemap.xml`
- Clear CloudFront cache
- Resubmit after 1 hour

### "Mobile usability issues"
- Test: https://search.google.com/test/mobile-friendly
- Should pass (already responsive)
- Fix any reported issues
