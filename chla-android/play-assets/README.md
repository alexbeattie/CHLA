# Google Play asset inventory

These files are the source-controlled first-release listing package for English
(United States). Upload the rendered files, not the SVG or master sources.

| Asset | File | Required format |
| --- | --- | --- |
| Store icon | `icon-512.png` | 512 x 512, 32-bit PNG with alpha, at most 1 MB |
| Feature graphic | `feature-graphic-1024x500.png` | 1024 x 500, 24-bit PNG without alpha |
| Phone screenshots | `screenshots/*.png` | 1080 x 2160, 24-bit PNG without alpha |
| English listing | `listing/en-US.md` | Name, short description, full description, reviewer notes |

The screenshots must be recaptured from the final API 36 release candidate after
every material UI change. Keep at least four accurate screens showing core app
experiences and do not add device frames, status notifications, or marketing
claims to the captures.
